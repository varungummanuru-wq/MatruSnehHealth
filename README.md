# MatruSneh Health 🤱

MatruSneh Health is a modern maternal healthcare Android application designed to support pregnant women with essential health tracking, reminders, and pregnancy care features.

The app provides an easy-to-use dashboard with healthcare tools, emergency assistance, nutrition guidance, and pregnancy monitoring features.

---

# ✨ Features

## 🍼 Pregnancy Care Dashboard
- Modern healthcare dashboard UI
- Easy navigation with feature cards
- Pregnancy progress tracking

## 👶 Kick Counter
- Track baby kicks
- Store kick history using Room Database
- View previous records

## 🍎 Nutrition Guide
- Pregnancy nutrition suggestions
- Healthy food guidance

## 💊 Medicine Reminder
- Set medicine reminders
- Notification support

## 📅 Appointment Reminder
- Track doctor appointments
- Reminder support

## ⚖️ BMI Calculator
- Calculate BMI during pregnancy
- Health awareness support

## 📈 Baby Growth Tracker
- Monitor baby growth stages

## 🚨 Danger Signs
- Important pregnancy warning signs
- Emergency awareness information

## 📊 Health Statistics
- Pregnancy-related tracking statistics

## ☎️ Emergency Support
- Quick emergency access section
